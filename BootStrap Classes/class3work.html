Different HTML Elements

An HTML element is defined by a start tag, some content, and an end tag.

Its important to know that these elements have attributes within their open tags that allow us to control the element. This includes inline CSS styling, adding a hyperlink, etc.


Styling

CSS stands for Cascading Style Sheets.
It saves a lot of work. It can control the layout
of multiple web pages all at once

  1. Inline CSS - Styles are given in the component itself
  2. Internal CSS - Styles are given in the head tag of the html seperatly
  3. External CSS - Styles are given from an external file that can be linked to out project

Some examples of CSS are adding a border, padding, margin, color, etc.
